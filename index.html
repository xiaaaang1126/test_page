AKM Estimation

Model Setup

\[ \ln w_{it} = \alpha_i + \psi_{J(i,t)}^{G(i)} + X_{it}'\beta + r_{it} \]

其中我們關注的參數為 (gender-specific) firm fixed effect, \(\psi_{J(i,t)}^{G(i)}\)，透過將餐飲及住宿業的公司視為 reference group，並計算其組內平均，再將所有的 firm fixed effect 扣除該平均後來做 normalization。另外，模型也加入年齡及其平方、三次方項 \((X_{it})\)。

Variance Decomposition

\[ \begin{aligned}Var(w_{it}) &= \underbrace{Var(\hat{\alpha}_i)}_{\text{Worker FE}} + \underbrace{Var\left(\hat{\psi}^{G(i)}_{J(i,t)}\right)}_{\text{Firm FE}} + \underbrace{2Cov\left(\hat{\alpha}_i, \hat{\psi}^{G(i)}_{J(i,t)}\right)}_{\text{Sorting}} \\&\quad + \underbrace{Var\left(X_{it}'\hat{\beta}^{G(i)}\right)}_{\text{Worker covariates}} + 2Cov\left(\hat{\alpha}_i + \hat{\psi}^{G(i)}_{J(i,t)}, X_{it}'\hat{\beta}^{G(i)}\right) + \underbrace{Var(\hat{r}_{it})}_{\text{Unexplained residual}}\end{aligned} \]

Premium Comparison

Premium Decomposition

Method

\[ \begin{aligned} &\overbrace{E[\psi^M_{J(i,t)} | \text{male}]}^{\text{Avg. Firm Premium for Male}} - \overbrace{E[\psi^F_{J(i,t)} | \text{female}]}^{\text{Avg. Firm Premium for Female}} &&\text{(a)} \\[0.3cm] =\quad& \underbrace{E[\psi_{J(i,t)}^M - \psi_{J(i,t)}^F \mid \text{male}]}_{\text{Bargaining Effect}} + \underbrace{E[\psi_{J(i,t)}^F \mid \text{male}] - E[\psi_{J(i,t)}^F \mid \text{female}]}_{\text{Sorting Effect}} &&\text{(b)}\\ =\quad& \underbrace{E[\psi_{J(i,t)}^M - \psi_{J(i,t)}^F \mid \text{female}]}_{\text{Bargaining Effect}} + \underbrace{E[\psi_{J(i,t)}^M \mid \text{male}] - E[\psi_{J(i,t)}^M \mid \text{female}]}_{\text{Sorting Effect}} &&\text{(c)} \end{aligned} \]

其中：

  • Sorting effect 代表「進入的公司不同所導致的薪資溢酬差距」（公司間的差距）。

    eq. (b) 計算女性的薪資溢酬，而 eq. (c) 則計算男性的薪資溢酬，兩者皆透過比較男女性的工作分佈來計算。

  • Bargaining effect 代表「在相同的公司工作，因為性別不同所導致的薪資溢酬差距」（公司內的差距）。

    eq. (b) 站在男性的公司來比較，而 eq. (c) 則是站在女性的公司來比較。

Inference

  • eq. (b) 是較為保守的估計（lower-bound estimates），儘管 bargaining effect 的結果可能被低估，大多文獻依舊採這個作為主要 interpret 的結果。

    • 道理很簡單：這裡的 bargaining effect 是去男性的公司計算 premium gap，但這些公司女性本來就難進入，因此會壓低 bargaining effect 的估計值。

  • eq. (c) 更為貼近女性的處境，但 bargaining effect 可能因為 reference group 不同而被放大。

    • 我們設定的 reference group 是「對女性沒有 rent 的低溢酬公司」（也就是餐飲業），雖然反映真實處境，但同時估計值也可能被放大。

Premium & Wage Gap

Sorting & Wage Gap

Bargaining & Wage Gap

  • 製造業（C）：premium gap 和其組成沒有太大改變但 wage gap 增加，代表在薪資差距因為 疫情/半導體產業 擴大的同時，firm premium distribution 沒有太大的改變。

  • 營建業（F）：premium gap 是負的而 wage gap 是正的，因為待在這個產業當中的女性有 selection bias，可能都於 premium 較高的公司中工作。

  • 批發及零售業（G）：premium gap 不變且 wage gap 縮小，使得解釋力增加，其中 sorting 增加而 bargaining 減少。可能因為 疫情 對薪資的負面影響導致薪資差距縮小。（？）

  • 金融與保險業（K）：premium gap 縮小而 wage gap 不變，並且主要來源是 bargaining effect 的下降。（？）

在 30 歲以下的群體，可以觀察到以下：

  • wage gap 都有擴大，但集中在 24 歲以下的樣本（沒讀學士、碩士）。
